# Unit Converter

Simple web-based unit converter project from Roadmap.sh.

- Project link: https://roadmap.sh/projects/unit-converter

## Run

Open `html/main.html` in your browser.

## Features

- Tab-based interface for `Length`, `Weight`, and `Temperature`
- Dynamic form rendering from a shared template
- Conversion logic split into dedicated modules:
	- `scripts/length.js`
	- `scripts/weight.js`
	- `scripts/temperature.js`
