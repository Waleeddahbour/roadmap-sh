# Number Guessing Game

CLI Number Guessing Game solution for Roadmap.sh.

- Project: https://roadmap.sh/projects/number-guessing-game

## Run

From this folder, run:

```bash
node number-guessing-game.js
```

## How it works

- Pick a difficulty:
	- Easy: 10 chances
	- Medium: 5 chances
	- Hard: 3 chances
- Guess a number between 1 and 100.
- The game gives hints after each guess and ends when you win or run out of chances.
